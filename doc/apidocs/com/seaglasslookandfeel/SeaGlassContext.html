Class SeaGlassContext

java.lang.Object
  javax.swing.plaf.synth.SynthContext
      com.seaglasslookandfeel.SeaGlassContext

public class SeaGlassContext
extends SynthContext

An immutable transient object containing contextual information about a Region. A SeaGlassContext should only be considered valid for the duration of the method it is passed to. In other words you should not cache a SeaGlassContext that is passed to you and expect it to remain valid.

Extends SynthContext by Scott Violet and reimplements most of its methods because too much is package local.
