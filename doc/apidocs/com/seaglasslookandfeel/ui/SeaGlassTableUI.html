setViewPortListeners

public static void setViewPortListeners(JTable table)
Adds striping to the background of the given JTable. The actual striping is installed on the containing JScrollPane's JViewport, so if this table is not added to a JScrollPane , then no stripes will be painted. This method can be called before the given table is added to a scroll pane, though, as a PropertyChangeListener will be installed to handle "ancestor" changes.

Parameters:
table - the table to paint row stripes for.
